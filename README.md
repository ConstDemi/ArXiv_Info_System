# ArXiv_Info_System
Вопросно-ответная система для arxiv.org. Итоговый проект ДПО.

• data - директория со всеми данными для проекта
• data/raw - скачанные "сырые" данные
• data/raw/tex - датасет из LaTeX исходников спарсенных статей
• data/raw/pdf - датасет из PDF-документов спарсенных статей
• data_parse.ipynb - Notebook для парсинга статей
• info.ipynb - Паспорт проекта
